What I've Learned

For starters, I've learned quite a bit about CSS this week. For instance, I've learned how to make this page quite a deal more appealing to the eye, and appear relatively more consistant across browsers. I've used an external CSS style sheet today, but you can use 2 other methods. You can also use a style section in your header and a style attribute to what you're manipulating. I've learned that the priority for what gets displayed when style sheets conflict is:

  1. Inline style.
  2. External and header style sections.
  3. Your browser's default style sheet.
According to w3schools css tutorial.
